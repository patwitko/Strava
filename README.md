# Marathon Data Analysis

This project focuses on analyzing marathon runners' performance data. The goal is to explore relationships between training metrics and race results.

## Key Features
* **Data Loading**: Processing the `MarathonData.csv` dataset.
* **Exploratory Data Analysis (EDA)**: Using `.head()`, `.describe()`, and `.info()` to understand the data structure.
* **Data Cleaning**: Identifying missing values with `.isna().sum()`.
* **Visualization**: Creating bar charts to visualize runner distributions and performance metrics.

## Tech Stack
* **Python**
* **Pandas**: For data manipulation and cleaning.
* **Matplotlib**: For generating insightful visualizations.

## Repository Content
* `analiza.ipynb` - Full Jupyter Notebook containing the analysis and charts.
